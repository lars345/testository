Git CheatSheet

befehle

allgemein

cd --> change directory

ls --> list

clone

git clone «ssh clone key» --> das repository clonen in die directory in welcher man sich befindet

git add * *=alles

git status

git commit -m “belibiger hinweis”

git push = um hochzuladen

branches

git branch --> anzeigen von allen branches

git branch «Name» --> neue branch hinzufügen

git checkout «Branchname» --> branch wechselny

git merge «branchname» --> man merged damit die angegebene branch mit der in welcher man sich befindet

git branch -d «branchname» --> eine Branch löschen
